# LeetCode Projects

A collection of Python solutions to LeetCode problems, built up as part of ongoing practice and interview preparation.

## Purpose

This repository serves as a practice log and portfolio piece while I prepare for software development roles. The goals are:

- **Sharpen problem-solving skills** through consistent, deliberate practice on algorithm and data-structure problems.
- **Maintain fluency in Python** — its idioms, standard library, and built-in data structures.
- **Prepare for technical interviews and employment skills tests** by working through the kinds of problems that commonly appear in screenings.
- **Document my reasoning**, often by including more than one solution per problem (for example, a brute-force approach alongside an optimized one) to make trade-offs in time and space complexity explicit.

Each problem lives in its own folder containing the original problem description, one or more solution scripts, and a problem-specific README explaining the approach and complexity analysis.
